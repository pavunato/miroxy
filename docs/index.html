miroxy

Micro Relay Proxy

A tiny, stateless proxy relay that runs on a Raspberry Pi. Forward any HTTP request to any API — one binary, zero config.

$ curl -fsSL https://miroxy.app/install.sh | sh

3rd Party API

Hide API keys, aggregate endpoints, proxy from the edge.

⚡

5MB Binary

Pure Go, zero dependencies. Runs on Raspberry Pi Zero 2 W with room to spare.

🔒

Token Auth

Optional bearer token to protect your proxy endpoint. Set one env var and you're done.

🚀

Stateless

No config files, no database. Send a POST with URL, headers, body — get the response back.

🌐

Full Control

Set method, headers, body, and timeout per request. Transform anything on the fly.

Usage

Request
curl -X POST http://localhost:8080/proxy \
  -H 'Content-Type: application/json' \
  -d '{
    "method": "GET",
    "url": "https://api.example.com/data",
    "headers": {
      "Authorization": "Bearer sk-xxx"
    },
    "timeout": 30
  }'
Response
// The upstream API response is streamed back directly
// Status code, headers, and body are all forwarded as-is
{
  "data": "from the 3rd party API"
}

Options

Flag / Env Description Default
--port Port to listen on 8080
--token Bearer token for auth —
MIROXY_ADDR Listen address (env var) :8080
